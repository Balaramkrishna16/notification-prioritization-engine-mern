# CYEPRO - Notification Prioritization Engine (MERN)

An intelligent system designed to categorize and prioritize incoming system notifications using a weighted scoring algorithm.

## 🚀 Tech Stack
- **Frontend:** Next.js 16, Tailwind CSS, Lucide React
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Authentication:** JWT (JSON Web Tokens)

## 🛠️ Features
- **Real-time Prioritization:** High, Medium, and Low urgency levels.
- **Smart Filtering:** Deduplication of repetitive system alerts.
- **Neural Link Dashboard:** Visualizing notification flow and system health.

## 📥 Installation
1. `npm install` (Root, Frontend, and Backend)
2. Create a `.env` file based on `.env.example`.
3. `npm run dev` to start the development server.
